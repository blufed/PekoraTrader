# PekoraTrader
PekoraTrader is a Trading tool for Korone/Pekora. It includes features such as Mass Trade Sender, Item/Player Lookup and a Blacklist feature.

To Install PekoraTrader you must follow the steps given under:

-Install the ViolentMonkey extension via https://violentmonkey.github.io/get-it/
-Next you must copy the full script provided in this repository and PASTE it in a new and empty ViolentMonkey Script.
-After you finish hit Save&Close on the top right.
-And next time you open Korone/Pekora the button to open the menu will be on the bottom right.
